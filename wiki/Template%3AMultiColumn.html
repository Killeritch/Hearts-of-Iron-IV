Usage

{{MultiColumn|

<!-- content here-->

|3|100px|10px}}

Where:

  • 3 = number of columns (optional; default)
  • 100px = width of columns (optional; default)
  • 10px = gap between columns (optional; default)


NB: Utilises CSS3; will only work on relatively new browsers.
